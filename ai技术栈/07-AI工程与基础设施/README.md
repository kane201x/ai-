## MLOps 成熟度模型

| 级别 | 特征 | 自动化程度 |
|------|------|-----------|
| L0：手工 | 脚本训练，手动部署 | 0% |
| L1：基础 | 实验跟踪，Docker 部署 | 30% |
| L2：流水线 | CI/CD for ML，自动重训 | 60% |
| L3：自动 | 自动特征/模型/超参 | 90% |
| L4：自适应 | 自动 A/B，漂移自恢复 | 100% |
